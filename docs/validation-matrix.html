Validation matrix.

Honest status tags for every major component. WORKING means wired and exercised; PARTIAL means code exists with known gaps; PLANNED is not shipped; RESEARCH is being evaluated.

WORKING code exists, registered, exercised. PARTIAL code exists but has gaps or external deps. PLANNED not shipped. RESEARCH being evaluated.

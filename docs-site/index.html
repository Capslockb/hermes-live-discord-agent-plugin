Hermes Live

A real-time, multimodal AI you can drop into any Discord voice channel. Open source, self-hostable, MIT.

v0.3.5 VOPI build sub-second latency 30+ voice tools multi-CLI fallback
status
VOPI build — first public functional release. The bridge is operational and shipping. Snappy interrupts (median 0.030ms), full multi-CLI fallback, email briefs, sfx library, video frame feeder, webhooks, and the full personality system are all in. A 6-finding code audit is published; the build is "functional, although it has rough edges." Stable enough for self-hosters; not yet feature-complete.

What's in the box

voice

Full-duplex audio

Sub-second latency. Discord UDP → Opus → 16 kHz mono → Gemini WSS. A 30ms local hard-clear VAD makes interrupts feel instant.

vision

Vision + frame feed

Send images or stream 1 fps screenshare. The model sees what you see — via a separate feeder on a host with a real display.

tools

Function calling

30+ voice tools. Calendar, mail, Home Assistant, GitHub, Spotify, files, search, and a multi-CLI delegation agent.

delegate

Multi-CLI fallback

opencode / codex / numasec / gemini / hermes-api with a health registry. If one is broken, work reroutes automatically.

proactive

Proactive notifications

Five channels: voice, DM, channel, webhook, auto. Fires on long-task completion, AFK pings, scheduled alerts.

mail

Email brief

Scheduled Gmail digest with 3-bucket importance scoring. AFK delivery via the notification dispatcher.

sfx

SFX library

Four slots — tool-init chime, error chain, notification ping, transition pop. Per-slot env overrides, lazy PCM cache.

memory

Honcho context

Per-user peer memory injected into the system prompt. The model remembers what you talked about last session.

control

Sidecar HTTP API

Local on 127.0.0.1:18943. /health, /frame, /say, /leave, /notify.

Get started

Five commands, two minutes.

# 1. Clone
git clone https://github.com/Capslockb/hermes-live-discord-agent-plugin.git
cd hermes-live-discord-agent-plugin

# 2. Install (prompts for env: DISCORD_BOT_TOKEN, GEMINI_API_KEY, user ID)
./install.sh

# 3. Restart the gateway
systemctl --user restart hermes-gateway

# 4. From Discord, in any voice channel:
/voice-live          # join
/voice-live-leave    # leave

# 5. Verify
curl -s http://127.0.0.1:18943/health | python3 -m json.tool
